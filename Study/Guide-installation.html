INSTALLATION ET CONFIGURATION DE L'APPLICATION WEB "WHIST"


  1. Installer un Web browser
    1. CHROME (conseillé!)
    2. FIREFOX
    3. BRAVE

  2. Installer le produit "Node.JS"
    1. NodeJS-WINDOWS
    2. NodeJS-MacOS

  3. Commandes POST installation de "Node.JS"

    • Depuis EXPLORER (Windows) ou FINDER (Mac):
      Créez un folder dont le nom "WHIST" est vivement conseillé.

    • Depuis EXPLORER (Windows) ou FINDER (Mac):
      Copiez le contenu "/dev" de la clef USB
      vers le folder "WHIST" précédemment créé.

    • Configureration application

      1. En premier lieu, depuis une fenêtre de commande...(*)
        controllez la bonne installation de:

        "Node.JS" via la commande: node -v
        "NPM" via la commande: npm -v

        Ces 2 commandes doivent vous retourner 2 numéros de version.

      2. Initialisation application JavaScript "WHIST"

        Commande: npm init -y

        Cette commande doit vous créer un fichier "package.json"

      3. Installation du module HTTP-WEB "EXPRESS"

        Commande: npm install EXPRESS

      4. Installation du module FILESYSTEM "FS"

        Commande: npm install FS

      5. Installation du module HTTP-LIST "SERVE-INDEX"

        Commande: npm install serve-index

      6. Installation du module HTTP-EXECUTION "NODEMON"

        Commande: npm nodemon G

    • Exécution application

      1. Depuis le folder "WHIST" démarrage "HTTP" en local:

        Commande: nodemon run web.js

      2. Accès à l'application "WHIST" via un web browser:

        Cliquez sur ce lien: http://localhost/

    • (*) PS: Fenêtre de commande ?

      • Comment ouvrir une fenêtre d'invite en fonction de l'OS
        pour y exécuter les différentes commandes de configuration:

        Windows: Windows/Key + R + 'cmd.exe' + [ENTER] key
        MacOS: Apple/Key + SPACEBAR + 'terminal.app' + [ENTER] key
