Petites recommendations avant de continuer...
Afin d'utiliser cette application de façon "efficace", il est conseillé:
  1. D'utiliser le navigateur "GOOGLE CHROME"
  2. Toutes les activités sur des fichiers (Lecture/Ecriture) s'opèrent depuis le folder défini dans votre navigateur
  3. Les données encodées sont sauvées dans la memoire de votre navigateur connu sous le nom de 'localStorage'.
    Il est important de savoir:
    1. Si vous changer de navigateur, les données sur l'un ne se retrouveront pas automatiquement sur l'autre.
      Ex: Vous êtes sur Firefox. Vous quittez Firefox pour ouvrir Chrome... vous ne verrez pas les données dans Chrome.
      Il vous faudra sauver les données dans un fichier depuis Firefox et les relire ensuite depuis Chrome.
    2. Quand vous nettoyez l'History de votre navigateur, sachez que vous pouvez tout perdre si vous avez sélectioné:
      Cookies and other data
